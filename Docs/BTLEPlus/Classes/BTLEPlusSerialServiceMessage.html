BTLEPlusSerialServiceMessage

@objc public class BTLEPlusSerialServiceMessage : NSObject

The BTLEPlusSerialServiceMessage class is the base class for all user defined messages that are sent over the serial service protocol. You pass instances of this to BTLEPlusSerialServiceController.send().
