BLEPlusSerialServiceController

@objc public class BLEPlusSerialServiceController : NSObject

BLEPlusSerialServiceController is controller that implements logic to send and receive data using the binary BLEPlus Serial Service protocol.

It’s implementation is independent of how the data is sent or received, leaving that up to the user.
