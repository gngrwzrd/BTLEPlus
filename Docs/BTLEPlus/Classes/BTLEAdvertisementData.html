BTLEAdvertisementData

@objc public class BTLEAdvertisementData : NSObject, NSCoding

The BTLEAdvertisementData class manages advertised data from Core Bluetooth. It can convert advertised data between it’s original discovered format, a plist supported format, and a serializable format with NSCoding. It provides some easier accessors to get and set advertisement data.
