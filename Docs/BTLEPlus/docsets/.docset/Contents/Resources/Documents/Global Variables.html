Global Variables

The following global variables are available globally.

  • BTLEPlusSerialServiceDefaultMTU

    The default maximum transmission unit. Upon connection, peers exchange their information and agree on an mtu and window size to use. It’s likely that the agreed upon mtu will be a lot less.

    Declaration

    Swift

    public let BTLEPlusSerialServiceDefaultMTU:UInt16 = UInt16.max
  • BTLEPlusSerialServiceDefaultWindowSize

    The default window size. Window size is the number of mtu buffers are available to send or receive. Total bytes in the window = windowSize * mtu.

    Declaration

    Swift

    public var BTLEPlusSerialServiceDefaultWindowSize:BTLEPlusSerialServiceWindowSize_Type = 32
  • BTLEPlusSerialServiceMaxWindowSize

    The largest possible window size. This is use to clamp user provided window sizes which are not allowed to be larger than this.

    Declaration

    Swift

    public var BTLEPlusSerialServiceMaxWindowSize:BTLEPlusSerialServiceWindowSize_Type = 128
  • BTLEPlusSerialServiceMaxMessageId

    The default max message id value before it loops to zero.

    Declaration

    Swift

    public var BTLEPlusSerialServiceMaxMessageId:BTLEPlusSerialServiceMessageId_Type = BTLEPlusSerialServiceMessageId_Type.max
