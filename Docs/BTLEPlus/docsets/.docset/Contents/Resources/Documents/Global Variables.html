Global Variables

The following global variables are available globally.

  • BLEPlusSerialServiceDefaultMTU

    The default maximum transmission unit. By default this is set to UInt16.max. The first thing that happens when peers are connected is exchanging peer info. When the central passes this value to the peripheral the peripheral is required to respond with either an ack, or a smaller size to use. From the peripheral size, CBCentral has a property called maximumUpdateValueLength that should be returned to the central as the mtu size.

    Declaration

    Swift

    public let BLEPlusSerialServiceDefaultMTU:UInt16 = UInt16.max
  • BLEPlusSerialServiceRealMTU

    This is the real payload mtu after headers are subtracted.

    Declaration

    Swift

    public let BLEPlusSerialServiceRealMTU:UInt16 = BLEPlusSerialServiceDefaultMTU - UInt16(BLEPlusSerialServicePacketProvider.headerSize + BLEPlusSerialServiceProtocolMessage.headerSize)
