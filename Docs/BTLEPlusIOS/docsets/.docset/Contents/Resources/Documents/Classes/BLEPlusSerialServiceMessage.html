BLEPlusSerialServiceMessage

@objc public class BLEPlusSerialServiceMessage : NSObject

BLEPlusSerialServiceMessage is the base class for sending user defined messages over the BLEPlus Serial Service.

The type property is automatically encoded for you in the transfer, on the peripheral or central side when you get a message callback you can inspect the type to figure out what message it is.
