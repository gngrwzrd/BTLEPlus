Global Variables

The following global variables are available globally.

  • BLEPlusSerialServiceDefaultMTU

    The default maximum transmission unit. Upon connection, peers exchange their information and agree on an mtu and window size to use. It’s likely that the agreed upon mtu will be a lot less.

    Declaration

    Swift

    public let BLEPlusSerialServiceDefaultMTU:UInt16 = UInt16.max
  • BLEPlusSerialServiceDefaultWindowSize

    The default window size. Window size is how many mtu buffers are available to send or receive. Total bytes in the window = windowSize * mtu.

    Declaration

    Swift

    public var BLEPlusSerialServiceDefaultWindowSize:BLEPlusSerialServiceWindowSize_Type = 32
  • BLEPlusSerialServiceMaxWindowSize

    The largest possible window size. This is use to clamp user provided window sizes which are not allowed to be larger than this.

    Declaration

    Swift

    public var BLEPlusSerialServiceMaxWindowSize:BLEPlusSerialServiceWindowSize_Type = 128
  • BLEPlusSerialServiceMaxMessageId

    The default max message id value before it loops to zero.

    Declaration

    Swift

    public var BLEPlusSerialServiceMaxMessageId:BLEPLusSerialServiceMessageId_Type = BLEPLusSerialServiceMessageId_Type.max
